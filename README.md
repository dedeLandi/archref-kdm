# archref-kdm

Para instalação e configuração de uso:

Ambiente de Programação: 
- Qualquer versão do Eclipse JEE (de preferencia a mais nova e estável)

Plugins que devem ser instalados obrigatoriamente:
- EMF - Eclipse Modeling Framework SDK
- MoDisco SDK (Incubation)
- Eclipse PDE Plug-in Developer Resources
- Eclipse RCP Plug-in Developer Resources

Plugins que devem ser atualizados:
- Eclipse Git Team Provider - Gitflow support
- Eclipse GitHub integration with task focused interface	

Plugins opcionais para instalação:
- Swing Designer
- Swing Designer Documentation	
- SWT Designer	
- SWT Designer Core	
- SWT Designer Documentation	
- WindowBuilder Core	
- WindowBuilder Core Documentation
- WindowBuilder Core UI	
- WindowBuilder GroupLayout Support	
- WindowBuilder Java Core	
- WindowBuilder XML Core (requires Eclipse WTP/WST)
